# Hiya Digital - Dynamic SEO Website

This project demonstrates a dynamic, SEO-optimized multi-country website built with Next.js.

## Features
- Dynamic country-based routing
- SEO-friendly URLs
- Country-based content switching
- Global navbar with active state
- Scalable architecture for multi-region websites

## Routes
- /india
- /germany
- /usa
- /contact/[country]
- /services/[country]
- /work-enquiry/[country]

## Tech Stack
- Next.js
- React
- Tailwind CSS
